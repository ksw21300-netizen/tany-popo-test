# Lotto Number Generator

## Overview

This project is a simple web application that generates and displays a set of 6 unique lotto numbers, ranging from 1 to 45. The application will have a clean, user-friendly interface with a button to generate new numbers.

## Features

*   Generate 6 unique random numbers.
*   Display the generated numbers in a visually appealing way.
*   Responsive design that works on different screen sizes.

## Project Structure

*   \`index.html\`: The main HTML file.
*   \`style.css\`: The CSS file for styling.
*   \`main.js\`: The JavaScript file for the application logic.
